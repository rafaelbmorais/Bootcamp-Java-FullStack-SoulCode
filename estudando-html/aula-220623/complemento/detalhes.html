O que é uma lista ordenada em HTML?

Uma lista ordenada em HTML é um elemento que permite criar uma lista de itens ordenados numericamente ou por meio de letras. É representada pela tag <ol>, que significa "ordered list" (lista ordenada). Cada item da lista é definido pela tag <li>, que significa "list item" (item da lista).

O que é uma lista não ordenada em HTML?

Uma lista não ordenada em HTML é um elemento que permite criar uma lista de itens não ordenados, ou seja, uma lista em que os itens são marcados com símbolos, como pontos ou traços. É representada pela tag <ul>, que significa "unordered list" (lista não ordenada). Cada item da lista é definido pela tag <li>, que significa "list item" (item da lista).

O que é uma lista de descrição em HTML?

Uma lista de descrição em HTML é um elemento utilizado para criar uma lista de termos e suas respectivas definições ou descrições. É representada pela tag <dl>, que significa "description list" (lista de descrição). Cada termo é definido pela tag <dt>, que significa "definition term" (termo de definição), e cada descrição é definida pela tag <dd>, que significa "definition description" (descrição de definição).

O que é uma âncora em HTML?

Uma âncora em HTML, também conhecida como hyperlink, é um elemento utilizado para criar links entre diferentes páginas da web ou dentro da mesma página. É representada pela tag <a>, que significa "anchor" (âncora).

A tag <a> possui um atributo obrigatório, chamado href, que especifica o destino do link, ou seja, o URL da página para a qual o usuário será direcionado ao clicar no link. Além disso, o texto dentro da tag <a> é exibido como o texto do link no navegador.
